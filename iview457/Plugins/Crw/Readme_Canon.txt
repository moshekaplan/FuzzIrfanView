- How to install CRW/CR2 DLLs?

- Unzip all files from the ZIP (including subfolders) into your "IrfanView" folder.
- Enable Canon CRW/CR2 DLLs in IrfanView: Properties->PlugIns.

-------------------------------------------------------------------------------
